# Phase 2 – Part 1: Data Generation and Import (PostgreSQL)

This folder contains Python scripts that generate SQL insert files and the resulting `insert_*.sql` scripts used to populate the project database.

## Folder Structure

- `dataset/`
  - Source CSV datasets provided for the project.
- `generating_scripts/`
  - Python generators that read the datasets and produce SQL insert scripts.
- `insert_sql/`
  - Final SQL files (e.g., `insert_product.sql`, `insert_customer.sql`, etc.) generated by the scripts.
- `dbkala.sql`
  - Database dump or schema file (if included).

## Requirements

- PostgreSQL 16+
- Python 3.10+
- Python packages:
  - `pandas` (if used by your scripts)
  - Standard library modules (`csv`, `json`, `hashlib`, etc.)
